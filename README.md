# Ateljé Sällström — Website

A modern, responsive website for Ateljé Sällström, a Swedish family art collective consisting of three artists: Lennart Sällström (father), Robin Sällström (son), and Ninni Sällström (daughter).

## Pages

- **Hem (Home)** — Hero section, introduction, artist previews, featured works
- **Galleri (Gallery)** — Filterable masonry grid with lightbox viewer
- **Om oss (About)** — Individual artist profiles with bios and exhibition history
- **Kontakt (Contact)** — Contact form and social media links

## Tech Stack

- Pure HTML5, CSS3, and vanilla JavaScript
- Google Fonts (Cormorant Garamond + DM Sans)
- Lucide Icons (CDN)
- Placeholder images from Unsplash

## Deployment

This is a static website ready for deployment. Simply upload the entire folder to any static hosting service:

### Netlify
1. Drag and drop the folder to [Netlify Drop](https://app.netlify.com/drop)
2. Or connect via GitHub and set the publish directory to the root

### Other Hosts
Upload all files maintaining the directory structure:
```
├── index.html
├── galleri.html
├── om-oss.html
├── kontakt.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── netlify.toml
└── README.md
```
